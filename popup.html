Click the symbol you would like to insert.

For faster usage you can filter by line and column. For example, opening this window and typing "1 11" will select the symbol at line 1 and column 11. Note that for example if you type "1 1" we can't know if you are done. So if you would like to select line 1 column 1 type "1 1 ". You can also open this window with Ctrl+Shift+Y.
